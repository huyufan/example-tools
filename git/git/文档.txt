删除远程分支
git branch -r -d origin/branch-name
git push origin:branch-name
查看本地分支
git branch
查看远程分支
git branch -a
创建分支
git branch test
切换分支
git checkout test
分支推到远程分支
git push origin test

git fetch --all
git reset --hard origin/master
git fetch 只是下载远程的库的内容，不做任何的合并 git reset 把HEAD指向刚刚下载的最新的版本
